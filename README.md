📚 Project-Software-Enginner-2

📌 Descripción
Este proyecto es una aplicación web desarrollada con Flask que permite gestionar un sistema de biblioteca. Los usuarios pueden registrarse, iniciar sesión y administrar información de libros.

🚀 Tecnologías utilizadas
* Python
* Flask
* SQLite
* HTML / CSS

📁 Estructura del proyecto
```
PROJECT-SOFTWARE-ENGINNER-2/
│── static/           # Archivos CSS y recursos
│── templates/        # Plantillas HTML
│── app.py            # Aplicación principal
│── init_db.py        # Script para inicializar la base de datos
│── database.db       # Base de datos SQLite
│── requirements.txt  # Dependencias del proyecto
│── README.md         # Documentación
```
🧠 Funcionalidades
* Registro de usuarios
* Inicio de sesión
* Gestión de libros (agregar, editar, eliminar)
* Base de datos SQLite
